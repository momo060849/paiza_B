# README

## プログラム概要
1行目の英大文字 X から、2行目の英大文字 Y の範囲に3行目のアルファベット C が含まれていれば"true", そうでなければ"false"と出力してください。
X が Y よりもアルファベット順で後ろになる場合(X = 'G', Y = 'F'のときなど)も"false"と出力してください。

## 入力される値
入力は以下のフォーマットで与えられます。
```
X
Y
C
```
入力値最終行の末尾に改行が１つ入ります。

入力値最終行の末尾に改行が１つ入ります。
文字列は標準入力から渡されます。

## 期待する出力
出力は、 "true" または "false" のどちらかです。

## 条件
```
・X, Y, Cはそれぞれ英大文字です。
```
